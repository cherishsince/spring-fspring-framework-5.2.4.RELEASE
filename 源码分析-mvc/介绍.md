###### HTTP



| GET     | 向Web服务器请求一个文件                  |
| ------- | ---------------------------------------- |
| POST    | 向Web服务器发送数据让Web服务器进行处理   |
| PUT     | 向Web服务器发送数据并存储在Web服务器内部 |
| HEAD    | 检查一个对象是否存在                     |
| DELETE  | 从Web服务器上删除一个文件                |
| CONNECT | 对通道提供支持                           |
| TRACE   | 跟踪到服务器的路径                       |
| OPTIONS | 查询Web服务器的性能                      |
